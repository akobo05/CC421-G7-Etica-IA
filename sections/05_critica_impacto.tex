\section{Crítica y evaluación de impacto}
La responsabilidad es tratar de definir qué considera apropiado el programador de software/científico, pero para Kant esta posibilidad de proponer una decisión de auto autónomo debe ser un acto moral según el imperativo categórico; en contraste, para Bentham estaría de acuerdo en utilizar un modelo matemático para definir un nivel de moralidad, no se negaría como Kant al decidir si el sistema comete un acto menos perjudicial. Mill posiblemente establecería un modelo donde el auto autónomo calcule en tiempo real cuáles son las implicaciones de una decisión, como qué tan responsables son los involucrados.
\section*{}Entonces, para el ingeniero de datos debería usar métricas tales como las consecuencias de una decisión, qué tan moral es la acción según el modelo de Bentham y si es moral o no, usando el Imperativo Categórico de Kant.
